# Sistema de Gestión de Estudiantes

## Descripción
Aplicación en Java que permite registrar y gestionar estudiantes desde consola.

## Funcionalidades
- Registrar estudiantes
- Listar todos los estudiantes
- Buscar estudiante por código
- Eliminar estudiante

## Tecnologías usadas
- Java 17
- Programación Orientada a Objetos (POO)
- ArrayList

## Conceptos aplicados
- Clases y objetos
- Encapsulación
- Constructor
- Getters y Setters
- ArrayList
- Validaciones

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir en Eclipse
3. Ejecutar Main.java
